**Create src/components/DealCustomizationForm.svelte:**

A Svelte 5 component with five branded input fields for deal customization. Uses `$state` for field values and `$derived` for validation.

Fields (all required):
1. **Deal Name** -- text input, placeholder "e.g., Project Atlas Acquisition", min 2 chars
2. **Parties Involved** -- text input, placeholder "e.g., Buyer Corp & Seller Inc.", min 2 chars
3. **Deal Type** -- `<select>` dropdown with options: "" (placeholder "Select deal type"), "M&amp;A", "IPO", "Debt Financing", "Equity Offering", "Private Placement", "Restructuring", "Other"
4. **Amount** -- text input, placeholder "e.g., $2.5 Billion", min 1 char
5. **Close Date** -- date input (`type="date"`)

Component interface using Svelte 5 `$props()`:
```typescript
interface Props {
  onchange?: (data: { attributes: Array<{key: string; value: string}>; isValid: boolean }) => void;
}
```

Use `$state` for each field. Use `$derived` to compute `isValid`: all five fields must be non-empty (trimmed). Deal type must not be empty string.

Use `$effect` to call `onchange` whenever fields change, passing:
- `attributes`: array of `{key, value}` objects: `[{key: "Deal Name", value: dealName}, {key: "Parties Involved", value: parties}, {key: "Deal Type", value: dealType}, {key: "Amount", value: amount}, {key: "Close Date", value: closeDate}]`
- `isValid`: boolean

Styling for each field group:
- Wrap each in `<div class="space-y-1.5">` with `<label class="block text-sm font-semibold uppercase tracking-wider text-navy">` and input with `class="w-full rounded-sm border border-navy/20 bg-white px-3 py-2.5 text-slate placeholder:text-navy/30 focus:border-gold focus:ring-1 focus:ring-gold focus:outline-none transition-colors"`.
- Select gets the same classes.
- Date input gets the same classes.
- Wrap all fields in `<div class="space-y-4">` with a heading `<h3 class="text-lg font-semibold text-navy mb-4">Customize Your Deal Toy</h3>`.
- Add a subtle note at bottom: `<p class="text-xs text-navy/50 mt-3">All fields are required for personalization.</p>`

**Update src/components/AddToCartForm.svelte:**

1. Import `DealCustomizationForm` from `./DealCustomizationForm.svelte`.
2. Add state: `let dealAttributes: Array<{key: string; value: string}> = $state([])` and `let isCustomizationValid = $state(false)`.
3. Add a handler function: `function handleCustomizationChange(data: { attributes: Array<{key: string; value: string}>; isValid: boolean }) { dealAttributes = data.attributes; isCustomizationValid = data.isValid; }`.
4. Render `<DealCustomizationForm onchange={handleCustomizationChange} />` ABOVE the existing form element (before the `<form>` tag).

7. In the `addToCart` function, pass attributes to `addCartItem`: change `addCartItem(item)` to `addCartItem({ ...item, attributes: dealAttributes })`.
