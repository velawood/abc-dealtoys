```css
    @apply inline-flex items-center justify-center;

    @apply transition-all duration-200 ease-in-out;

  }

  .button-outline {
    @apply inline-flex items-center justify-center;
    @apply border-2 border-gold text-gold bg-transparent font-semibold text-sm tracking-wider uppercase;
    @apply px-8 py-3 rounded-sm;
    @apply transition-all duration-200 ease-in-out;
    @apply hover:bg-gold hover:text-navy;
    @apply focus-visible:ring-2 focus-visible:ring-gold focus-visible:ring-offset-2 focus-visible:outline-none;
    @apply disabled:cursor-not-allowed disabled:opacity-60;
  }
```

IMPORTANT:
- Remove the OLD `.button` class that uses `bg-gradient-to-br from-emerald-900` — replace it entirely with the new branded version above.
- The .button class MUST be inside `@layer base`, not a separate @layer.
- Use `rounded-sm` (not `rounded-lg`) for a more premium/editorial feel consistent with the serif typography.
- Use `tracking-wider uppercase` for button text to match premium brand conventions.
- `hover:bg-gold/85` (85% opacity) provides a subtle darkening effect.
- `hover:shadow-lg` adds depth on hover.
- Both button variants must have `transition-all duration-200 ease-in-out` for smooth micro-interactions.
