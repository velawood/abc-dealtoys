**1. Create `src/components/MobileNav.svelte`:**

A Svelte 5 component for the mobile navigation overlay. This follows the existing pattern (CartIcon.svelte uses `client:load`).

```svelte
<script lang="ts">
  let isOpen = $state(false);

  function toggle() {
    isOpen = !isOpen;
  }

  function close() {
    isOpen = false;
  }
</script>
```

Template structure:
- A hamburger button (`md:hidden`, min-w-[44px] min-h-[44px] for touch target, aria-label="Open menu")
- When `isOpen`: a full-screen overlay (fixed inset-0 z-50 bg-navy) with close button and nav links stacked vertically
- Nav links: Shop (/products), How It Works (/how-it-works), Our Story (/about), Get a Quote (/contact)
- Link styling: text-white text-2xl font-medium, with hover:text-gold transition-colors duration-200
- Each link calls `close()` on click to dismiss the overlay
- The hamburger icon: 3 horizontal bars using spans with bg-current, styled in text-white (on navy header)
